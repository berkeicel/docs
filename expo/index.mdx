## Quick Start

<Steps>
  <Step title="Install the SDK">
    ```bash
    npx expo install rampkit-expo-dev react-native-webview react-native-pager-view react-native-root-siblings
    ```
  </Step>
  <Step title="Initialize RampKit">
    ```typescript
    import { RampKit } from "rampkit-expo-dev";

    await RampKit.init({
      appId: "YOUR_APP_ID",
      onOnboardingFinished: (payload) => {
        navigation.replace("Home");
      },
    });
    ```
  </Step>
  <Step title="Show Onboarding">
    ```typescript
    RampKit.showOnboarding();
    ```
  </Step>
</Steps>

## Main API

```typescript
import { RampKit } from "rampkit-expo-dev";

// Initialize (required)
await RampKit.init({ appId: "YOUR_APP_ID" });

// Show/hide onboarding
RampKit.showOnboarding();
RampKit.closeOnboarding();

// Check state
RampKit.isInitialized();
RampKit.getUserId();
RampKit.getDeviceInfo();

// Track events
RampKit.trackEvent("custom_action", { key: "value" });
RampKit.trackScreenView("screen_name");
RampKit.trackCtaTap("button_id");

// Cleanup
await RampKit.cleanup();
```

## Native APIs

```typescript
import { Haptics, StoreReview, Notifications } from "rampkit-expo-dev";

// Haptics
await Haptics.impactAsync("medium");
await Haptics.notificationAsync("success");

// Store Review
await StoreReview.requestReview();

// Notifications
const result = await Notifications.requestPermissionsAsync();
```
