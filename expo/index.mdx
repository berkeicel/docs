## Install the SDK

Add `rampkit-expo` or `rampkit-react-native` with npm or yarn.

## Initialize RampKit

One line setup using `RampKit.init({ apiKey })`.

## Load Your First Flow

Render a live onboarding with `RampKit.loadFlow('onboarding_id')`.

## Preview Mode

Test unpublished drafts instantly during development.
