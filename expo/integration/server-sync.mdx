---
title: "Server-Side User Sync"
description: "Sync RampKit user attributes with your backend"
---

```ts
const attrs = RampKit.getUserAttributes();
await fetch("https://api.yourbackend.com/rampkit-sync", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify(attrs),
});
```
