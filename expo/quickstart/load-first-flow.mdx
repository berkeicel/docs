---
title: "Show Onboarding"
description: "Display your onboarding flow"
---

After initialization:

```typescript
RampKit.showOnboarding();
```

## Full Example

```typescript
import { useEffect } from "react";
import { Button, View } from "react-native";
import { RampKit } from "rampkit-expo-dev";

export default function App() {
  useEffect(() => {
    RampKit.init({
      appId: "YOUR_APP_ID",
      onOnboardingFinished: () => {
        console.log("Done!");
      },
    });
  }, []);

  return (
    <View style={{ flex: 1, justifyContent: "center", alignItems: "center" }}>
      <Button title="Start" onPress={() => RampKit.showOnboarding()} />
    </View>
  );
}
```

## Auto-Show

To show onboarding automatically after init:

```typescript
await RampKit.init({
  appId: "YOUR_APP_ID",
  autoShowOnboarding: true,
  onOnboardingFinished: () => {
    navigation.replace("Home");
  },
});
```
