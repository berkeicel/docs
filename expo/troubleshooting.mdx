## Quick Fixes

### Onboarding Not Showing

```typescript
// 1. Check initialization
console.log("Initialized:", RampKit.isInitialized());
console.log("Data loaded:", !!RampKit.getOnboardingData());

// 2. Ensure init completed before showing
await RampKit.init({ appId: "YOUR_APP_ID" });
RampKit.showOnboarding();
```

### Events Not Tracking

```typescript
// Enable debug mode
await RampKit.init({
  appId: "YOUR_APP_ID",
  environment: "development",
});

// Check logs for [RampKit] EventManager: Event sent:
```

### Variables Not Syncing

```javascript
// In WebView, use correct format:
window.ReactNativeWebView.postMessage(JSON.stringify({
  type: 'rampkit:variables',
  vars: { key: 'value' }
}));
```

### Native Module Error

```bash
# Reinstall native dependencies
npx expo prebuild --clean
npx pod-install
```

## Debug Mode

Enable verbose logging:

```typescript
await RampKit.init({
  appId: "YOUR_APP_ID",
  environment: "development",
});
```

Filter logs with `[RampKit]` prefix.
