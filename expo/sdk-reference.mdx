## Core Methods

| Method | Description |
|--------|-------------|
| `RampKit.init(config)` | Initialize the SDK |
| `RampKit.showOnboarding()` | Show the onboarding |
| `RampKit.closeOnboarding()` | Close the onboarding |
| `RampKit.isInitialized()` | Check if ready |
| `RampKit.getUserId()` | Get user ID |
| `RampKit.getDeviceInfo()` | Get device info |
| `RampKit.cleanup()` | Clean up resources |

## Event Tracking

| Method | Description |
|--------|-------------|
| `RampKit.trackEvent(name, props)` | Track custom event |
| `RampKit.trackScreenView(name)` | Track screen view |
| `RampKit.trackCtaTap(id)` | Track button tap |
