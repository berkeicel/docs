---
title: "Expo SDK — Reference"
description: "RampKit API for Expo"
---

- `RampKit.init(config)` — Initialize once at app startup.
- `RampKit.loadFlow(id, options)` — Fetch and render a flow.
- `RampKit.setUserAttributes(data)` — Pass app data into flows.
- `RampKit.getVariables()` / `setVariable(key, value)` — Manage flow variables.
- `RampKit.on(event, callback)` / `off(event)` — Subscribe to flow events.
- `RampKit.reset()` — Clear variables and cached sessions.
