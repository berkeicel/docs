The RampKit SDK provides a comprehensive API for onboarding, analytics, and native features.

## Main SDK

```typescript
import { RampKit } from "rampkit-expo-dev";
```

### Core Methods

| Method | Description |
|--------|-------------|
| `init(config)` | Initialize the SDK |
| `showOnboarding(opts?)` | Display the onboarding overlay |
| `closeOnboarding()` | Dismiss the onboarding overlay |
| `cleanup()` | Release SDK resources |

### State Accessors

| Method | Returns |
|--------|---------|
| `isInitialized()` | `boolean` |
| `getUserId()` | `string \| null` |
| `getDeviceInfo()` | `DeviceInfo \| null` |
| `getOnboardingData()` | `any` |

### Event Tracking

| Method | Description |
|--------|-------------|
| `trackEvent(name, properties, context?)` | Track custom event |
| `trackScreenView(screenName, referrer?)` | Track screen view |
| `trackCtaTap(buttonId, buttonText?)` | Track button tap |

## Native APIs

```typescript
import { Haptics, StoreReview, Notifications } from "rampkit-expo-dev";
```

### Haptics

| Method | Description |
|--------|-------------|
| `impactAsync(style)` | Trigger impact haptic |
| `notificationAsync(type)` | Trigger notification haptic |
| `selectionAsync()` | Trigger selection haptic |

### Store Review

| Method | Description |
|--------|-------------|
| `requestReview()` | Request in-app review |
| `isAvailableAsync()` | Check if review is available |
| `hasAction()` | Check if action is supported |

### Notifications

| Method | Description |
|--------|-------------|
| `requestPermissionsAsync(opts?)` | Request notification permissions |
| `getPermissionsAsync()` | Get current permission status |

## Utilities

```typescript
import { 
  getRampKitUserId,
  collectDeviceInfo,
  buildRampKitContext,
  eventManager

```

| Function | Description |
|----------|-------------|
| `getRampKitUserId()` | Get stable user ID |
| `collectDeviceInfo()` | Collect device information |
| `buildRampKitContext(deviceInfo)` | Build template context |
| `getSessionDurationSeconds()` | Get current session duration |
